See Soia in Action

Define once, use everywhere with type-safe generated code.

from soiagen import shapes

# Create type-safe objects
point = shapes.Point(x=3, y=4, label="P")

# Serialize to JSON
json_data = shapes.Point.SERIALIZER.to_json(point)

# Deserialize back
restored_point = shapes.Point.SERIALIZER.from_json(json_data)

# Type safety guaranteed
assert(restored_point == shapes.Point(x=3, y=4, label="P"))
import { Point, Serializer } from './generated/shapes';

// TypeScript types are generated automatically
const point: Point = {
  x: 3,
  y: 4,
  label: "P"
};

// Full type checking at compile time
const json = Serializer.toJson(point);
const restored = Serializer.fromJson(json);
#include "shapes.h"

// C++ with full type safety
auto point = shapes::Point{
  .x = 3,
  .y = 4,
  .label = "P"
};

// Efficient binary serialization
auto binary = shapes::Point::serialize(point);
auto restored = shapes::Point::deserialize(binary);

Get Started in Minutes

Install the compiler and start building type-safe APIs.

1

Install the Compiler

npm install -g soiac
2

Define Your Schema

// api.soia
struct User {
  id: int64;
  name: string;
  email: string;
}
3

Generate Code

soiac --python --typescript api.soia
